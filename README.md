# DjangoGirls-Template
This is my first project using the popular Django framework, we coded in some languajes as Python, HTML and some QuerySets<br /><br />

## How to setup this repository in local

### Install Git
First of all, you need to have git installed on your computer, you can download it from [here](https://git-scm.com/downloads)

### Clone this repo
From inside a terminal like bash or cmd you need to enter this command inside the folder you want to set this repo:
```bash
  git clone https://github.com/joseluissaiz/DjangoGirls-Template.git
```

<br /><br />
## How to start Django server
From your terminal, go inside the DjangoGirls-Template folder and then use this command to enter the virtual environment:
```bash
  myenv\Scripts\activate
```
Then, you should have entered, now, to run the server you need to invoke this method:
```bash
  py manage.py runserver
```
<br /><br />
## Stop service
To exit the server, just intoduce [CTRL]+[C] and it will stop.
